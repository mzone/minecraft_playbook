# 🔨 Minecraft はじめてガイド　クラフトへん
## 〜 サバイバル はじめのいっぽ 〜

---

## 🌟 クラフト（Crafting）ってなに？

あつめた **Materials（マテリアルズ）＝ ざいりょう** をつかって、
ものをつくることだよ！

### ✅ この じゅんばんで すすもう！

```
① Wood をあつめる
② Planks と Stick をつくる
③ Crafting Table をつくる
④ 木のどうぐをつくる
⑤ Stone をほる
⑥ 石のどうぐをつくる
⑦ Furnace をつくる
⑧ Iron を手に入れる
```

---

## ① 🪵 Wood（ウッド）＝ 木のみき をあつめよう

> 🤲 **てで** たたけば こわせるよ！まず 10こくらいあつめよう。

---

## ② 🪵 Planks（プランクス）＝ きのいた をつくろう

```
[ Wood ]
```

> Wood 1こ → Planks 4まい になるよ。てのなかでつくれる！

---

## ③ 🪵 Stick（スティック）＝ ぼう をつくろう

```
[ Planks ]
[ Planks ]
```

> Planks 2まい → Stick 4ほん になるよ。てのなかでつくれる！

---

## ④ 🔨 Crafting Table（クラフティングテーブル）をつくろう！

> これをつくると **3×3 のますめ** がつかえて、いろんなものがつくれるよ！

```
[ Planks ][ Planks ]
[ Planks ][ Planks ]
```

> 🪵 **Planks × 4** でつくれるよ！

---

## ⑤ 🪨 木のどうぐ（Wooden Tools）をつくろう

> まず **Wooden Pickaxe** をつくって Stone をほれるようにしよう！

### ⛏️ Wooden Pickaxe（ウドゥン ピックアックス）＝ 木のツルハシ

```
[ Planks ][ Planks ][ Planks ]
[        ][ Stick  ][        ]
[        ][ Stick  ][        ]
```

> 🪵 Planks × 3 ＋ Stick × 2

---

### 🪣 Wooden Shovel（ウドゥン シャベル）＝ 木のシャベル

```
[        ][ Planks ][        ]
[        ][ Stick  ][        ]
[        ][ Stick  ][        ]
```

> 🪵 Planks × 1 ＋ Stick × 2

---

### 🪓 Wooden Axe（ウドゥン アックス）＝ 木のオノ

```
[ Planks ][ Planks ]
[ Planks ][ Stick  ]
[        ][ Stick  ]
```

> 🪵 Planks × 3 ＋ Stick × 2

---

### ⚔️ Wooden Sword（ウドゥン ソード）＝ 木のけん

```
[        ][ Planks ]
[        ][ Planks ]
[        ][ Stick  ]
```

> 🪵 Planks × 2 ＋ Stick × 1

> ⚠️ **Wooden Sword はよわい！** でもさいしょはこれでまもろう。すぐ Stone Sword にしよう！

---

## ⑥ 🔥 Furnace（ファーネス）＝ かまど をつくろう！

> Iron（てつ）をとかしたり、たべものをやくのにつかうよ！

```
[ Cobblestone ][ Cobblestone ][ Cobblestone ]
[ Cobblestone ][             ][ Cobblestone ]
[ Cobblestone ][ Cobblestone ][ Cobblestone ]
```

> 🪨 **Cobblestone × 8** でつくれるよ！
> Cobblestone は **Wooden Pickaxe** で Stone をほるとでてくるよ。

### 🔥 Furnace でできるもの（序盤）

| いれるもの（Fuel＝ねんりょうも いれてね） | できるもの |
|---|---|
| Iron Ore（アイアンオア）＝ 鉄鉱石 | 🔩 Iron Ingot（アイアンインゴット） |
| Wood / Log（ウッド）| 🖤 Charcoal（チャコール）＝ すみ |
| Raw Beef / Pork（ローミート）| 🍖 Cooked Meat（クックドミート） |

> 💡 **Fuel（フューエル）＝ ねんりょう** には **Wood・Planks・Charcoal・Coal** がつかえるよ！

---

## ⑦ 🪨 石のどうぐ（Stone Tools）をつくろう

> 木よりつよい！Stone のどうぐにしたら Iron もほれるよ！

### ⛏️ Stone Pickaxe（ストーン ピックアックス）＝ 石のツルハシ

```
[ Cobblestone ][ Cobblestone ][ Cobblestone ]
[             ][ Stick       ][             ]
[             ][ Stick       ][             ]
```

> 🪨 Cobblestone × 3 ＋ Stick × 2

---

### ⚔️ Stone Sword（ストーン ソード）＝ 石のけん

```
[        ][ Cobblestone ]
[        ][ Cobblestone ]
[        ][ Stick       ]
```

> 🪨 Cobblestone × 2 ＋ Stick × 1

---

## ⑧ 💡 Torch（トーチ）＝ たいまつ をつくろう！

> あんいところをあかるくするよ。たくさんつくっておこう！

```
[        ][ Coal / Charcoal ]
[        ][ Stick           ]
```

> ⚫ **Coal または Charcoal × 1** ＋ **Stick × 1** → **Torch × 4 こ** できるよ！

---

## ⑨ 🛏️ Bed（ベッド）＝ ベッド をつくろう！

> ねると **夜をとばせる**よ！よるは Zombie（ゾンビ）がでるから、はやくつくろう！

```
[ Wool   ][ Wool   ][ Wool   ]
[ Planks ][ Planks ][ Planks ]
```

> 🐑 **Wool（ウール）× 3**（おなじいろ）＋ 🪵 **Planks × 3**
> Wool は **Sheep（シープ）＝ ひつじ** を **Shears（ハサミ）** か **てで** たおしてとろう。

---

## ⑩ 🎒 Chest（チェスト）＝ はこ をつくろう！

> アイテムをしまっておけるよ。いえのそばにおいておこう！

```
[ Planks ][ Planks ][ Planks ]
[ Planks ][        ][ Planks ]
[ Planks ][ Planks ][ Planks ]
```

> 🪵 **Planks × 8** でつくれるよ！

---

## 📌 ざいりょう かんたんまとめ

| えいご（English） | よみかた | どうやってとる？ |
|---|---|---|
| Wood / Log | ウッド / ログ | 🤲 てで木をたたく |
| Planks | プランクス | Wood からつくる |
| Stick | スティック | Planks 2まいからつくる |
| Cobblestone | コブルストーン | ⛏️ Pickaxe でいしをほる |
| Coal | コール | ⛏️ Pickaxe で Stone の中のくろいものをほる |
| Charcoal | チャコール | 🔥 Furnace で Wood をやく |
| Iron Ingot | アイアンインゴット | 🔥 Furnace で Iron Ore をとかす |
| Wool | ウール | 🐑 ひつじをたおす or ハサミでとる |
