# タスク: レゴ風のステップバイステップ組み立てマニュアル画面の作成

## デザイン要件
- レゴ（LEGO）の公式組み立て説明書をオマージュしたUIにしてください。
- 全体的に文字（テキスト）による説明は極力排除し、ビジュアルで直感的に理解できるデザインにします。
- カラーパレット: 背景はクリーンな白（#FFFFFF）または薄いグレー。強調色にはレゴらしい黄色（#FFD400）や赤を使用。
- フォント: 丸みのある太めのサンセリフ。

## 機能要件（コンポーネント構造）
1. **LegoManualContainer**: 全体の状態（現在のStep数）を管理する親コンポーネント。
2. **PartsRequiredBox**: 画面左上に配置。「そのステップで新しく使うブロックの形状」と「数量（x2 などのテキスト）」を黄色い枠線の中に表示。
3. **MainBuildingStage**: 画面中央。現在のステップの完成図（SVG画像、またはCSSで作ったアイソメトリックなブロックの図）を表示。
4. **StepNavigation**: 画面下部。大きな「前へ」「次へ」ボタンと、現在のステップ数（例: 04 / 12）を表示。

## 追加要件
- ブロックが合体するステップでは、新しく追加されるパーツの周りに**黄色い矢印（⬇）や点線**を表示して、どこにハメ込めばいいか視覚的にわかるようにすること。

## 実装の手順
- 上記構造のコンポーネントを Next.js + Tailwind CSS で作成すること。
- ステップを「次へ」と進めると、中央の画像と必要なパーツの表示が切り替わるように、モックデータ（3ステップ分ほど）を用意してシミュレートすること。
