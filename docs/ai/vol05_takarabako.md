## p.2 — つかうもの（よういするもの）

**イラスト構成**:
アイテムアイコンを格子状に並べる。各アイコンの下に数字。

```
[ いた ]     ×64+  ← 採掘（木から）
[ チェスト ] ×6    ← 「つくる」マーク（いた×8で1こ）
[ かんばん ] ×6    ← 「つくる」マーク（いた×6+スティック×1）
[ スティック ] ×2  ← 「つくる」マーク
```

「つくる」マーク：ハンマーアイコン

**テキスト**:
- 「つかうもの」（上部タイトルのみ）
- 補足：「いたは たくさん つかうよ。き を たくさん あつめよう」

## p.21 — おうちのひとへ（親向けページ）

### この巻で体験すること

| 操作 | 方法 |
|---|---|
| チェストを開く | 右クリック |
| アイテムを移動 | クリックして移動先でクリック |
| スタックを一括移動 | Shift+クリック |
| 看板を設置 | 右クリック（地面や壁） |

### 難易度：★☆☆☆☆

### 所要時間の目安
- じゅんびへん（素材集め〜クラフト）：15〜20分
- つくろうへん（建築）：10〜15分
- あそび（整理整頓）：10〜20分（楽しくて長くなることが多い）

### つまずきやすいポイント

**①チェストが開けられない**
チェストの真上に別のブロックがあると開けられません。
上部に1ブロック以上の空間を確保してください。
また、ラージチェスト（チェスト2個を横に並べると大容量に）も紹介すると喜ぶ子が多いです。

**②アイテムの移動方法**
アイテムをチェストに移すには：
- 1個ずつ：クリック（手に持つ）→ チェスト内でクリック（置く）
- スタック全部：Shiftクリックで一括移動（子どもにはこれを教えると早い）
- 分けて置く：右クリックで1個ずつ分けられる

**③看板の文字が書けない**
テキスト入力は保護者が手伝ってください。カテゴリ名は子どもと一緒に考えると
「自分のお店」感が出て愛着がわきます。

### この巻の教育的ポイント
- **分類・整理の思考**：どのカテゴリに入れるか考えることで分類能力が育まれます
- **記憶と検索**：どこに何を入れたか覚えて後で取り出す体験
- **達成感の可視化**：1〜18巻で集めたアイテムが整理されることで「冒険の全体像」が見える
