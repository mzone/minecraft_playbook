## p.18 — おうちのひとへ（親向けページ）

### この巻で体験すること

| 操作 | 方法 |
|---|---|
| 馬に乗る（なつかせ） | 右クリック（素手） |
| 馬のインベントリを開く | 乗った状態でEキー |
| くらを装着 | インベントリでくらスロットにドラッグ |
| 馬で移動 | WASDキー |
| 馬でジャンプ | スペースキー |
| 馬から降りる | 左Shiftキー |

### 難易度：★★★☆☆

### 所要時間の目安
- じゅんびへん（柵クラフト〜サドル入手）：15〜20分
- つくろうへん（牧場建設〜馬なつかせ）：15〜20分
- あそび（乗馬）：10分〜

### 新発見：「馬のなつかせ方」について
馬は何も持たずに右クリックすると乗ろうとしますが、
なつき度が不十分だと弾かれます。繰り返すことでなつき度が上がり、
最終的にハートエフェクトが出てなつきます。
なつき度はランダム要素があり、平均3〜7回程度かかります。

### サドル（くら）の入手について（重要）
サドルはクラフトできません。以下の方法で入手できます：

1. **村のチェスト**：村の中の建物（特に鍛冶屋・図書館・武器屋）のチェスト
2. **砂漠の神殿（Desert Temple）**：砂漠バイオームにある大きなピラミッド状の建物の地下チェスト（罠あり、必ず保護者が操作）
3. **廃坑チェスト**：地下の廃坑（Mineshaft）のチェスト
4. **クリエイティブで取得**：最も確実。一時的にクリエイティブに切り替えてサドルを取得後、サバイバルに戻す

### つまずきやすいポイント

**①馬が見つからない**
馬は平原バイオームや草原に自然スポーンします。
スポーンしていない場合は少し離れた場所まで探索するか、保護者がクリエイティブで馬をスポーンさせてあげてください（スポーンエッグ使用）。

**②何度乗っても弾かれる**
なつき度が上がるのに時間がかかるお子さんもいます。「何回弾かれるか数えよう」とゲーム感覚にすると挫折しにくいです。

**③馬に乗ったままゲートを出られない**
ゲートを開けるには乗ったまま右クリックできます。お子さんには少し難しいので保護者が補助してあげてください。

**④馬が牧場から逃げる**
ゲートを閉め忘れると馬が出てしまいます。「乗った後はとびらをしめよう」と習慣づけてください。
