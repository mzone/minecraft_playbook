## p.2 — つかうもの（よういするもの）

**目的**: 材料の全量提示。アイコン＋数字のみ。文字はほぼ不要。

**イラスト構成**:
アイテムアイコンを格子状に並べる。各アイコンの下に数字。

```
[ まるた ]   ×8
[ ものづくりだい ]  ×1  ← 「つくる」マーク付き
[ ドア ]     ×1  ← 「つくる」マーク付き
```

「つくる」マーク：ハンマーアイコンなど、「自分で作る必要がある」ことを示す記号

**テキスト**:
- 「つかうもの」（上部タイトルのみ）
- 各アイテム名はひらがなで小さく（まるた／ものづくりだい／ドア）

**AI向け補足**:
- まるた＝原木（Oak Log）。木を壊すと手に入る。
- ものづくりだい＝クラフティングテーブル。まるたから作る。
- ドア＝木のドア（Oak Door）。ものづくりだいで作る。

### p.10 — くりかえして いたを あつめよう

**アクション**: いたが30枚になるまで繰り返す

**イラスト**:
- まるた（8個）がいた（32枚）に変換される全体図
- 「まるた 8こ → いた 32まい」を大きく図解

**ひとこと**: 「30まい あつめよう」

残り分をまるたから追加で変換すればよい。初回は「30まいめやす」でOK。

## p.21 — おうちのひとへ（親向けページ）

### この巻で体験すること

| 操作 | 方法 |
|---|---|
| 木を壊す | 左クリック長押し |
| もちものを開く | Eキー |
| ブロックを置く | 右クリック |
| ドアを開ける | 右クリック |

### 難易度：★☆☆☆☆

### 所要時間の目安
- じゅんびへん（材料集め〜クラフト）：15〜20分
- つくろうへん（建築）：10〜15分
- あそび：10分〜

### つまずきやすいポイント

**①まるたを拾い忘れる**
壊した後に光るアイテムが地面に落ちます。「近づいて踏む」動作が必要です。
拾えていないときは、もちもの画面を一緒に確認してください。

**②クラフトのマスが分からない**
インベントリの2×2マスと、ものづくりだいの3×3マスは別物です。
2×2でドアは作れません（レシピが3マス使うため）。
画面を一緒に見ながら指差しサポートをおすすめします。

**③入口のあけ方（どこを壊す？）**
南の壁の中央を「縦2マス」壊す必要があります。
1マスだけ壊してもドアが入りません。「2つ こわそう」と声かけを。

**④ドアの向きがおかしい**
右クリックする位置（上/下のマス）によってドアの開く向きが変わります。
向きが気にならない年齢なら気にしなくてOKです。

### ピースフルモードについて
モンスターが出ないため安全です。空腹は自動回復するので食料管理も不要です。
ただし高いところから落ちると体力が減ります。建築中は落下に注意してください。
