**テキスト**:

**AI向け補足**:

- タワー側面（高さ4マス）にはしごを1列4枚貼る。
- ものづくりだい：このシリーズは各巻スタンドアロンのため、毎回作成を想定。

### p.7 — いたを えだに かえよう

**ページ番号**: じゅんびへん B-2
**アクション**: いた2枚を縦に並べてえだ（スティック）に変換する

**イラスト**:
- 2×2インベントリクラフト欄（左列の上下2マスにいた）
- 「いた 2まい → えだ 4ほん」大きな矢印で図解
- 「たてに ならべる」を矢印で強調

```
[いた][　]     →   えだ×4
[いた][　]
```

**howTo**:
1. Eキーでもちものを開く
2. クラフト欄の左上のマスに「いた」を1まい置く
3. その真下のマスに「いた」をもう1まい置く（たて2マス）
4. 右に「えだ×4」が出てきたらクリック（タップ）して受け取る
5. 「えだ」が14ほん以上になるまでくりかえそう

**ひとこと**: 「たてに ならべよう」

**必要数（AI向け補足）**:
- はしご2回クラフトにスティック14本必要
- いた2枚→えだ4本なので4回クラフト（いた8枚消費）で16本できる

**イラスト**:

- H字型にえだを配置（7本）
- 配置を黄色ハイライトで強調

```
[えだ][　][えだ]
[えだ][えだ][えだ]
[えだ][　][えだ]
```

**howTo**:
1. ものづくりだいに近づいて右クリック/タップして開く
2. 3×3クラフト欄の左上にえだを置く

4. 真ん中の行の3マスすべてにえだを置く
5. 左下にえだを置く
6. 右下にえだを置く（合計7本：H字型）
7. 結果欄の「はしご×3」をクリック/タップして受け取る
8. もう一度同じ手順でもう3まい作る（合計6まい以上）

**ひとこと**: 「Hのかたちに ならべよう」

**AI向け補足**:

## p.20 — おうちのひとへ（親向けページ）

### この巻で体験すること

| 操作 | 方法（Java） | 方法（Bedrock/Switch） |
|---|---|---|
| 木を壊す | 左クリック長押し | 画面タップ長押し |
| もちものを開く | Eキー | …ボタン（中央） |
| ブロックを置く | 右クリック | 画面タップ |
| はしごに設置する | 右クリック（壁面に向けて） | タップ（壁面に向けて） |
| はしごを登る | Wキー押し続け | 左スティック前倒し |
| はしごを降りる | Sキーまたは何もしない | スティックを離す／後ろ倒し |

### 難易度：★★☆☆☆

### 所要時間の目安
- じゅんびへん（材料集め〜クラフト）：15〜20分
- つくろうへん（建築）：10〜15分
- あそび：10分〜

### つまずきやすいポイント

**①はしごを空中に置こうとする**
はしごはブロックの壁面にしか設置できません。「タワーのかべに むかって おこう」と
声かけしてください。タワーに体を近づけて、壁面を向いた状態で右クリックが必要です。

**②のぼる方向キーが分からない**
「まえに進むキー」を押し続けることでのぼります。
Javaは「W」、Switchは「左スティックを前」、タッチは「上ボタン」です。
はしごにきちんと体が触れていないとのぼれません。

**③まるたを積む向きがずれる**
原木は向きがあります（縦・横）。縦に積むには、上から置くと縦向きになります。
美しさより子どもが自分で置けることを優先してください。

**④高さから落下する**
ピースフルモードでも高所から落ちると体力が減ります（高さ4ブロック以上で落下ダメージ）。
てっぺんでは慎重に移動するよう声かけを。体力が減ったら食べ物を食べて回復します。

### ピースフルモードについて
モンスターなし・空腹自動回復。落下ダメージのみ注意が必要です。
テーブルは各巻スタンドアロン設計のため、1巻のひみつきちがなくても進められます。
