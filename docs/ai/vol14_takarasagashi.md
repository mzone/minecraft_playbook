## p.22 — おうちのひとへ（親向けページ）

### この巻で体験すること

| 操作 | 方法 |
|---|---|
| 地面・石を掘る | 左クリック長押し |
| ツルハシを持ち替える | 1〜9キー / マウスホイール |
| たいまつを壁に置く | ブロック面に右クリック |
| クラフト | ものづくりだいを右クリック |

### 難易度：★★★★☆

### 所要時間の目安
- じゅんびへん（ツルハシ・たいまつクラフト）：15〜20分
- つくろうへん（入口〜トンネル〜鉱石採掘）：20〜25分
- あそび（さらに掘る・集める）：10分〜

### 新発見：「鉱石採掘」について
この巻はシリーズの中で最も複雑な内容です。クラフトを2回（木のツルハシ→石のツルハシ）行い、
地下採掘という新しい空間体験をします。
鉱石を見つけたときの驚きと達成感が非常に大きいので、
「何が掘れたか一緒に確認する」時間を設けてあげてください。

### ツルハシの素材と採掘可能な鉱石の関係

| ツルハシ | 採掘できる主な鉱石 |
|---|---|
| 木（Wooden） | 石炭鉱石・銅鉱石 |
| 石（Stone） | 鉄鉱石・金鉱石・エメラルド・ラピスラズリ |
| 鉄（Iron） | ダイヤモンド・レッドストーン |
| ダイヤ・ネザライト | すべて |

この巻では石のツルハシまで作成し、鉄鉱石の採掘を体験します。

### つまずきやすいポイント

**①木のツルハシで鉄鉱石を掘ろうとする**
木のツルハシで鉄鉱石を掘ると、ブロックが壊れてもアイテムがドロップしません（消えてしまいます）。
「橙色の石は石のツルハシで！」と声かけを。
間違えて掘っても危険はないので安心してください。

**②暗い場所が怖い**
地下は暗いため怖がるお子さんもいます。
「たいまつを先に置いてから進む」習慣を教えてあげてください。
ピースフルモードなのでモンスターは出ません。

**③トンネルが迷路になって出られない**
5ブロック程度の短いトンネルであれば問題ありませんが、
深く掘りすぎると出口が分からなくなることがあります。
「出口の近くにたいまつを目印に置く」か、
迷子になった場合はインベントリを開いて/spawnpointコマンドでリスポーン地点に戻る方法を保護者が把握しておいてください。

**④落下ダメージ**
地下に深く掘り進めると、穴に落ちてダメージを受けることがあります。
ピースフルモードでは体力は自然回復しますが、高所からの落下は注意。

**⑤丸石と石炭鉱石の見分け**
丸石は石を掘るとドロップするグレーのブロック。石炭鉱石はインベントリ画面で確認可能（黒い模様）。
4〜6歳には「黒い点々があるかどうか」で見分けるよう伝えてください。

### この巻の後の次のステップ（任意）
- 鉄鉱石をかまどで精錬すると鉄インゴットになる（かまどクラフト）
- 鉄インゴットで鉄ツルハシを作るとさらに深い鉱石が掘れる
- 次シリーズへの布石として「もっと深いところに何があるか」を親子で話し合うのもおすすめ

### 特記：断面図の使用
地下トンネルは等角投影図に加え、断面図（横から見た図）を多用する。
トンネルの深さや構造が直感的に伝わるよう、
断面インセット（小窓）を各ページ右下に配置することを推奨。
