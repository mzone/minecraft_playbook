## p.2 — つかうもの（よういするもの）

**イラスト構成**:
```
[ つるはし ]    ×1   ← 「つくる」マーク付き
[ いし ]        ×8   ← 「ほりだす」マーク付き
[ かまど ]      ×1   ← 「つくる」マーク付き
[ まるた ]      ×2   ← 「もえるもの」マーク（ねんりょうに つかう）
[ にわとり ]    →    ← 「さがす」マーク（なまどりにく を もらう）
[ やきとり ]    ×1   ← 完成品
```

**テキスト**:
- 「つかうもの」（上部タイトルのみ）

**AI向け補足**:
- かまどのレシピ：3×3グリッドの外周8マスに Cobblestone（コブルストーン）= かまど1個
  中央は空けること（チェストと同じ外周パターン）。
- つるはし（Pickaxe）のレシピ：3×3の上段3マスにいた（または石）＋中段中央・下段中央にえだ
  実際は: いた×3（上段）＋えだ×2（縦2マス中央）= 木のつるはし×1
- 石（Cobblestone）は木のつるはしで掘ると効率的。素手でも壊れるが遅い。
- ニワトリ（Chicken）を倒すと生のとり肉（Raw Chicken）が落ちる。ピースフルでも動物は倒せる。

## p.20 — おうちのひとへ（親向けページ）

### この巻で体験すること

| 操作 | 方法 |
|---|---|
| 石を掘る | 左クリック長押し（つるはし持参） |

| アイテムをスロットへ | クリックして移動 |
| 食べる | 右クリック長押し（食べ物を手に持って） |

### 難易度：★★☆☆☆

### 所要時間の目安
- じゅんびへん（採掘〜クラフト）：15〜20分
- つくろうへん（調理）：10〜15分
- あそび：10分〜

### つまずきやすいポイント

**①石炭鉱石と普通の石を見間違える**
石炭鉱石（Coal Ore）は灰色の石に「黒い点々」があります。
見分けにくい場合は「くろい てんてんが ある！」と声かけしてあげてください。

**②つるはしがなくて石が掘れない**
素手でも石は壊れますが、時間がかかります。まずつるはしを作りましょう。
「きのつるはし」（木材で作るつるはし）でも石は掘れます。

**③かまどUIの操作が分からない**
上のスロット（食材）と下のスロット（燃料）を間違えやすいです。
「したに もやすもの、うえに やくもの」と繰り返し声かけすると覚えやすいです。

**④食べられない（満腹ゲージが満タン）**
ピースフルモードでは空腹になりにくいです。
少し走り回って空腹ゲージを下げてから食べるよう伝えてください。

### この巻の教育的ポイント
- **変換の概念**：食材が「熱を加えると変わる」体験は現実の料理の理解につながります
- **燃料と焼くものの分離**：2つの材料（燃料と食材）を分けて管理する体験
