## p.2 — つかうもの（よういするもの）

**イラスト構成**:
```
[ まるた ]    ×3
[ えだ ]      ×6   ← 「つくる」マーク付き
[ せきたん ]  ×6   ← 「ちかで ひろう」マーク付き
[ たいまつ ]  ×6   ← 「つくる」マーク付き
[ ものづくりだい ] ×1  ← 「つくる」マーク付き
```

**テキスト**:
- 「つかうもの」（上部タイトルのみ）

**AI向け補足**:
- せきたん＝Coal。石（Cobblestone）を掘ると「せきたんのこうせき」（Coal Ore）が見つかる。
  地下に行かなくても山の崖や地面から1〜2ブロック掘ると見つかる。
  木炭（Charcoal）でも代替可能：まるたをかまどで焼くと木炭になる（4巻で詳しく学ぶ）。
- たいまつのレシピ：えだ1本（下）＋せきたん1個（上）= たいまつ4個（2×2インベントリクラフト可）
- えだのレシピ：いた2枚（縦）= えだ4本


## p.20 — おうちのひとへ（親向けページ）

### この巻で体験すること

| 操作 | 方法 |
|---|---|
| 石を掘る | 左クリック長押し |
| もちものを開く | Eキー |
| たいまつを壁に設置 | 右クリック（壁面に向けて） |
| ブロックを壊す | 左クリック長押し |

### 難易度：★★☆☆☆

### 所要時間の目安
- じゅんびへん（石炭集め〜クラフト）：15〜20分
- つくろうへん（設置）：10〜15分
- あそび：10分〜

### つまずきやすいポイント

**①石炭が見つからない**
地表から2〜3ブロック掘ると石炭鉱石（黒い点のある石）が見つかります。
山の崖面でも見えることがあります。見つからない場合は「木炭」で代替できます（4巻で学ぶかまどを先に体験しても）。

**②たいまつが壁に設置できない**
たいまつは「ブロックの面に向けて」右クリックする必要があります。
壁に近づいて壁面を見た状態で右クリックしてください。
地面に置く場合は、地面を向いて右クリックします。

**③まどの穴から子どもが出てしまう**
穴のサイズが1×1なら通り抜けられません（1.5ブロックの高さが必要）。
安全面から、まどは高さ2段目（地面から2マス目）に開けるのがお勧めです。
