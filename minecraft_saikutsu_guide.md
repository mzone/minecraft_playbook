# ⛏️ Minecraft はじめてガイド　さいくつへん

## 🌟 さいしょに

マインクラフトでは、じぶんでものをほったり、あつめたりして、いろんなものをつくれるよ！
つかうどうぐによって、とれるものがちがうよ。じょうずにつかいわけよう！

---

## 🛠️ どうぐ と とれるもの いちらん

### ⛏️ Pickaxe（ピックアックス）＝ ツルハシ

> かたいものをほるどうぐ。いしやこうせきにつかおう！

| えいご（English） | よみかた | どんなもの？ |
|---|---|---|
| Stone | ストーン | ふつうのいし |
| Coal | コール | くろいこうせき。たいまつをつくれる |
| Iron | アイアン | にぶいぎんいろのこうせき |
| Gold | ゴールド | きいろいこうせき |
| Diamond | ダイヤモンド | あおくひかるこうせき |
| Cobblestone | コブルストーン | ほったあとのごつごつした石 |
| Netherrack | ネザーラック | ネザーにある赤いいし |

---

### 🪣 Shovel（シャベル）＝ シャベル

> やわらかいものをほるどうぐ。つちやすなにつかおう！

| えいご（English） | よみかた | どんなもの？ |
|---|---|---|
| Dirt | ダート | ふつうのつち |
| Sand | サンド | さばくや海ぞいにあるすな |
| Gravel | グラベル | じゃり。たまにFlint（くろよし）がでる |
| Snow | スノー | ゆきのせかいにある |
| Clay | クレイ | かわのそこにある白っぽいつち |
| Soul Sand | ソウルサンド | ネザーにあるこわい顔のすな |

---

## 📌 まとめ

```
⛏️ Pickaxe（ツルハシ） → いし・こうせき
🪣 Shovel（シャベル）  → つち・すな・じゃり
🪓 Axe（オノ）        → 木・もくざい
🌾 Hoe（クワ）        → はたけ・葉っぱ
✂️ Shears（ハサミ）   → ウール・草・つる
🤲 Hand（て）         → はな・きのこ
```

> 💡 **ポイント！**
> ただしいどうぐをつかうと、**はやくほれる**し、**ぜんぶとれる**よ！
> まちがったどうぐでほると、おそくてとれないこともあるよ。

---

## 🎮 つぎは…

あつめたものをつかって **Crafting（クラフティング）**＝ ものづくりをしよう！
